# 📚 Book Collection App

A modern React application for managing your personal book collection. Built with Vite and Tailwind CSS.

## ✨ Features

- 📖 View a list of books with covers
- ➕ Add new books to your collection
- ❌ Delete books from your collection
- ✅ Mark books as read/unread
- 📊 View statistics (total, read, unread)
- 🎨 Beautiful responsive design with Tailwind CSS

## 🚀 Live Demo

[View Live Demo](https://book-collection-khaki.vercel.app)

- ⚛️ **React** - UI library
